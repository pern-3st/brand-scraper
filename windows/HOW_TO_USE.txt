BRAND SCRAPER - HOW TO USE
==========================

FIRST TIME ONLY
---------------
1. Make sure you are on Windows 10 or 11 and connected to the internet.
2. Open the "brand_scraper" folder.
3. Open the "windows" sub-folder.
4. Double-click "setup.bat".
5. If Windows shows a blue "Windows protected your PC" box, click
   "More info", then click "Run anyway". This is normal for unsigned
   scripts and the file is safe.
6. A black window will open. Press any key when prompted.
7. Wait 5-10 minutes while it installs everything.
   (If Windows asks "Do you want to allow this app to make changes?", click Yes.)
8. When you see "Setup complete!", press any key to close the window.

EVERY TIME YOU WANT TO USE IT
-----------------------------
1. Open the "windows" sub-folder.
2. Double-click "run.bat".
   (Same "Windows protected your PC" prompt may appear -
    click "More info" then "Run anyway".)
3. Two black windows will open - leave them alone.
4. Your web browser will open the app automatically.
5. Use the app in your browser.

WHEN YOU ARE DONE
-----------------
- Either double-click "stop.bat", OR
- Close the two black windows labelled "Brand Scraper Backend"
  and "Brand Scraper Frontend".

IF SOMETHING GOES WRONG
-----------------------
- Close everything and double-click "run.bat" again.
- If it still does not work, message <YOUR NAME HERE> and screenshot
  any error message you see.

DO NOT
------
- Do not delete, move, or rename any files in this folder.
- Do not edit the file called ".env" inside the "backend" folder -
  it contains a key that lets the app work.
